# nodejs-web-api-express-graphql-hello-world

## Description
A basic web api that uses graphql.

## Tech stack
- nodejs

## Docker stack
- node:latest

## To run
`sudo ./install.sh -u`
- http://localhost

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
